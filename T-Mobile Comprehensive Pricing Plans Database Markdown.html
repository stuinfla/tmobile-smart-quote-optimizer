üí∞ TAX BREAKDOWN & AUTOPAY DETAILS

Florida Communications Tax Structure

Base Plan Price: $90 (Example)
+ Florida Communications Tax (7.44%): $6.70
+ Local Sales Tax (7% avg): $6.30  
+ Regulatory Fees: $3.99/line
= Total Monthly Tax/Fees: ~$16.99/line

AutoPay Discount Details

  • Postpaid Plans: $10/month discount per line (most plans)
  • Prepaid Plans: $5/month discount per line
  • Eligible Payment Methods: Bank account (ACH) or debit card
  • Credit Cards: Not eligible for AutoPay discount

Tax Exemptions

  • Military: May qualify for tax exemptions on base
  • Government: Special government rates available
  • Business: Different tax structure applies

Database Fields

  • plan_id: Unique identifier
  • plan_name: Display name
  • plan_type: postpaid/prepaid/connect
  • pricing: Nested object with line counts
  • features: Object with feature flags
  • discounts: Array of available discount types
  • availability: Current availability status

API Endpoints Suggestion

GET /api/v1/plans/{plan_id} - Specific plan
GET /api/v1/plans/pricing/{lines} - Pricing for X lines
GET /api/v1/plans/calculate-tax/{zip} - Tax calculator
